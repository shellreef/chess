The Alfaerie Plus images were downloaded from http://www.chessvariants.com/graphics.dir/alfaerie-plus/ for this Alfaerie Omega package.

The following images are included:

  • dababbarider
  • elephantmodern - this was renamed from elephant, since alfaerie already includes an elephant image for alfil (Parlett: ~2X), while this one is for the modern elephant (Parlett: 2X).
  • elephantridermodern - renamed from elephantrider
  • highpriestess
  • nwd
  • parallelgeneral
  • templar2

The following images were already included in other sets, so they won't be shown here:

  • flexibleknight
  • jumpinggeneral (as jumpinggeneral2)
  • slidinggeneral (as slidinggeneral2)
  • templar (as templar2)
  • twistedknight
  • zigzaggeneral (as zigzaggeneral2)
