alfaeriemisc unofficial package

This file lists the images contained at http://www.chessvariants.org/graphics.dir/alfaeriemisc/ which have been integrated into the Alfaerie Omega package. The majority of the images have been integrated into the flat structure without any renaming, but several images (listed below) had to be renamed to avoid conflicts. Some images were not included because they were already included by another expansion pack.
