Alfaerie Omega Chess Variant Image Package

This directory contains graphics from the excellent Alfaerie Chess Variant Graphics set, plus expansion sets. The goal is to include all known Alfaerie chess graphics into one, flat archive. Instead of having separate downloads to add new images, a new version of this package can be released when there are new images. Contributions are welcome! Note that I didn't make most of these images, I'm mereley collecting and organizing them.

Included here:

  • Original Alfaerie Variant Chess Graphics Set (source)
  • Alfaerie Expansion Set One (source)
  • Alfaerie Expansion Set Two (source)
  • Alfaerie Expansion Set Three (source)
  • Alfaerie Expansion Set Four (source)
  • There is no http://www.chessvariants.com/graphics.dir/alfaerie6/ yet..
  • Alfaerie Plus (source) is included here, with a few minor modifications necesssary to avoid name conflicts (details)

There are some more graphics available at: http://www.chessvariants.org/graphics.dir/alfaeriemisc/ which are not yet included. TODO: include!

In general, images are named beginning with b or w, for blue or white, followed by a descriptive name for the piece. In some cases, g, r, or k prefixes are seen for green, red, or black. However, the blue pieces are intended to be used for the black pieces in chess. Only transparent gif images are included here, not Windows bitmaps.
